Documentation and development

How to create documentation for this project

  1. Install Doxygen.
  2. Edit Doxyfile published in this folder and change paths according to the location of your local repository.
  3. Copy Doxyfile to Doxygen's bin folder.
  4. To generate html documentation files, execute from doxygen\bin folder, the command
Doxygen Doxyfile
